# Rootscape

Ein Point-and-Click-Rechenspiel zum Thema Wurzeln (Escape-Room-Prinzip).

Du und deine Katze wollt raus – aber die Tür lässt sich nicht öffnen.
Löse die Wurzel-Aufgaben an den Stationen im Raum, um zu entkommen. 10 Minuten Zeit!

**Spielen:** https://manuel-benz.github.io/rootscape/

## Entwicklung

```bash
npm install
npm run dev
```

Gebaut mit React, Vite und Tailwind CSS. Deployment auf GitHub Pages erfolgt automatisch per GitHub Actions bei jedem Push auf `main`.
